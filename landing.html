FAQ

Häufige Fragen

Brauche ich technisches Wissen? ▾
Nein. SaunaManager ist komplett webbasiert — kein Download, keine Installation. Einfach URL aufrufen und loslegen. Alles ist so einfach wie eine normale Website.
Funktioniert die App auf allen Geräten? ▾
Ja — PC, Mac, iPad, iPhone, Android-Tablet. Alle Geräte die einen modernen Browser haben. Die Daten werden automatisch synchronisiert.
Können mehrere Mitarbeiter gleichzeitig arbeiten? ▾
Ja. Mit dem Starter-Plan können bis zu 3 Geräte gleichzeitig auf dieselben Daten zugreifen und diese bearbeiten.
Sind unsere Daten sicher? ▾
Alle Daten werden verschlüsselt auf deutschen Servern gespeichert (IONOS). Passwortgeschützter Zugang, HTTPS-Verschlüsselung, regelmäßige Backups.
Kann ich die App kostenlos testen? ▾
Ja — der Free-Plan ist dauerhaft kostenlos. Keine Kreditkarte, keine Kündigung nötig. Einfach registrieren und loslegen.
Gibt es eine App für das Handy? ▾
Es gibt keine native App — aber die Website funktioniert perfekt auf dem Handy. Sie können sie auf dem Homescreen speichern und sie verhält sich wie eine echte App.

Bereit für professionelles Aufguss-Management?

Starten Sie heute kostenlos. Keine Kreditkarte, keine Kündigung, keine versteckten Kosten.
